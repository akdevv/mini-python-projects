# Rock Paper Scissors

## Description
Python Rock Paper Scissors game in terminal. User v/s Computer (which chooses at random) with cool ASCII arts.

## How to Use
1. **Clone the Repository:**
    ```bash
    git clone <repository-url>
    cd band-name-generator
    ```
2. **Run _main.py_:**
    ```bash
    python main.py
    ```
