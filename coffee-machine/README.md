# Coffee Machine

## Overview
This Python project is an Object-Oriented Programming implementation of a coffee machine.

## How to Use
1. **Clone the Repository:**
    ```bash
    git clone https://github.com/akdevv/mini-python-projects.git
    cd mini-python-projects
    cd coffee-machine
    ```
2. **Run _main.py_:**
    ```bash
    python main.py
    ```
