# Higher Lower

## Overview
This Python project is a "Higher or Lower" guessing game where the player attempts to predict whether the next randomly selected celebrity will be higher or lower than the current one in terms of followers.

## How to Use
1. **Clone the Repository:**
    ```bash
    git clone https://github.com/akdevv/mini-python-projects.git
    cd mini-python-projects
    cd higher-lower
    ```
2. **Run _main.py_:**
    ```bash
    python main.py
    ```
